      Add a filter function into a filter group

      addFilter(string $filterName, string $filterFunc) 
      If the group does not exists, it will be automatically created. The function must only have one argument to accept the data passed through it by applyFilter() method and then return the modified content. Example usage: $filter->addFilter("emoticon", "replaceEmoticon"); function replaceEmoticon(&$content) { // code to replace all :D, :^",etc into image return $content; }

      Parameters

      $filterName

      string

      Filter group's name

      $filterFunc

      string

      Filter function's name

      Passthrough content into all filter functions of a group.

      applyFilter(mixed $content, string $filterName) 
      How the content is modified is based on the behavior of each filter function. Example Usage: // Replace all emoticon inside $content into image $filter->applyFilter($content, "emoticon");

      Parameters

      $content

      mixed

      The content variable which needs to be filtered. Can be of any type.

      $filterName

      string

      The filter group's name
